Component Reference

Complete guide to all KodNest Premium Build System components. Each component maintains the design philosophy of calm, intentional, coherent, and confident.

Buttons

Primary Button

Main call-to-action. Solid deep red background with white text.

<button class="btn btn-primary">Action</button>
Secondary Button

Alternative action. Outlined with deep red border and text.

<button class="btn btn-secondary">Alternative</button>
Tertiary Button

Less prominent action. Transparent with text only.

<button class="btn btn-tertiary">Less Important</button>
Button Size: Small

Use for compact layouts or secondary buttons.

<button class="btn btn-primary btn-sm">Small</button>
Button Size: Large

Use for prominent actions or highlight important CTAs.

<button class="btn btn-primary btn-lg">Large</button>
Button: Disabled State

When action is unavailable. Reduced opacity, cursor not-allowed.

<button class="btn btn-primary" disabled>Disabled</button>

Neutral

<span class="badge badge-neutral">Neutral</span>
